The harmony of JavaScript and windows.

The popular desktop operating system paradigm, now in the browser!

The web is the new emerging platform for developers worldwide, and it is starting to catch up to the ease-of-use and the styling capabilities of modern desktop operating systems.

And as with most modern desktop operating systems, a popular desktop paradigm has been the window. Versatile, fast and easy to manage, the window is perhaps one of the most important GUI innovations. And now it's coming to the browser.

What is JSwm?

JSwm is a window manager for the browser.

The JavaScript Window Manager, or JSwm for short, provides a way for web developers to utilise common windowing paradigms without having to go through the tedious process of creating it yourself. It can also double as a fancy, and very easily customisable, browsing experience.

What can you do with it?

JSwm has a variety of frameworks built from the ground up to be as user-friendly and as dependable as possible. You can get all the features that you get from a regular desktop window manager, such as:

  • A desktop, complete with program icons and shortcuts
  • Windows, manipulated with the mouse and keyboard
  • Menu bar, with all open windows, and minimised ones too
  • Ability to maximise, minimise and close your windows
  • Keep-on-top functionality
  • Workspaces, switchable through shortcut or menu bar widget
  • Various window templates, such as dialog boxes, tree views and more
